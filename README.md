# MyCardApp

Intro to iOS App Development Learn how to make an iPhone app using Apple's Swift and SwiftUI


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Usage/Examples

```Swift
import SwiftUI

@main
struct appwithgoogledevApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}

```


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
